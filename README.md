# Personal Portfolio Website

A clean and fully responsive personal portfolio website built from scratch to showcase my projects and background.

**Live Demo:** [benhiggins.uk](https://benhiggins.uk)

## Features
- **Dark/Light Mode:** Theme toggle with persistent user preference storage (saved in `localStorage`).
- **Interactive Contact Form:** Fully functional contact form integrated with the Web3Forms API and a custom success redirect page.
- **Fully Responsive:** Mobile friendly layout using modern CSS Flexbox.

## Built With
- HTML5 (Semantic markup)
- CSS3 (Custom properties/variables, flexbox)
- JavaScript (ES6+) (DOM manipulation, theme handling, form validation resets)
- Web3Forms API (Serverless form handling)
- Cloudflare & GitHub Pages (Hosting and DNS management)
